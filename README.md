# 🎞 Scheduler-Project

### **Outline**
프로젝트 플래너 관리를 위한 웹사이트

### ⏱ **Period**
- 2023-06-24 ~

### 🛠 **Environment**
- Language: Java 17
- Framework : Spring boot 3.0.8
- IDE: IntelliJ
- Build Tool: Gradle
- DB: MariaDB
- Template Engine: JSP

### 📄 **Key Features**
- 프로젝트 플래너 관리
- 프로젝트 플래너에 대한 스케줄링
- 커뮤니티
- 사용자의 게시물에 대한 이미지 파일 처리
- 스케줄링을 통한 장기간 미접속 회원에 대한 배치 처리

### 🔧 **Key Skills**
- Schedule
- Spring Batch 5.0
- Spring Security
- Dynamical Generation of HTML Element & JQuery Event Handling
