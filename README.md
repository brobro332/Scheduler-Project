# 🎞 Scheduler-Project

### **Outline**
일정관리를 위한 웹사이트

### ⏱ **Period**
- 2023-06-24 ~

### 🛠 **Environment**
- Language: Java 17
- Framework : Spring boot 3.0.8
- IDE: IntelliJ
- Build Tool: Gradle
- DB: MariaDB
- Template Engine: JSP

### 📄 **Key Features**
- To Do List 관리(추후 변경될 수 있음)
- 주간일정관리(추후 변경될 수 있음)
- 등등
- 사용자페이지와 관리자페이지로 나뉨
